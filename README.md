# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Vikash12Singh/Vikash12Singh.github.io
```

Navigate into the project

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
